# C++ Projects

A collection of beginner-to-intermediate C++ projects demonstrating core programming concepts, problem-solving techniques, and object-oriented programming. This repository will continue to grow with more C++ applications and utilities.

## 📂 Projects

### 1. 🧮 Calculator

A simple command-line calculator application built in C++. It performs basic arithmetic operations through an interactive menu.

#### Features

* Addition
* Subtraction
* Multiplication
* Division
* User-friendly menu
* Handles multiple calculations in a single session

#### Concepts Used

* Functions
* Conditional Statements (`if` / `switch`)
* Loops
* User Input & Output
* Basic Arithmetic Operations

---

### 2. 💻 Mini OS

A console-based Mini Operating System simulation developed in C++. It demonstrates how common operating system utilities can be implemented in a simple terminal application.

#### Features

* Menu-driven interface
* File and directory related utilities
* Basic system operations
* User interaction through console
* Modular implementation using functions/classes

#### Concepts Used

* Object-Oriented Programming (OOP)
* Functions
* Classes & Objects
* File Handling
* Loops & Conditional Statements
* Modular Programming

---

## 🛠️ Requirements

* C++ Compiler (GCC, G++, MSVC, Clang, etc.)
* C++11 or later

---

## 🚀 How to Compile

### Using g++

```bash
g++ filename.cpp -o output
./output
```

### Example

```bash
g++ calculator.cpp -o calculator
./calculator
```

```bash
g++ mini-os.cpp -o mini-os
./mini-os
```
